# 1.1 API 摘要

所有接口均使用现有 JWT，并按当前用户验证课程、资料、计划、任务或图谱资源。

## Agent 与引用

- `POST /api/agent/chat`
- `POST /api/agent/chat/stream`
- `GET /api/agent/sessions/{session_id}/messages`

assistant 消息的 `citations` 为真实检索结果；`agent_trace` 为独立执行过程。

## 综合规划

- `POST /api/study-plans/multi/preview`
- `POST /api/study-plans/multi`
- `POST /api/study-plans/{plan_id}/multi/preview-regeneration`
- `POST /api/study-plans/{plan_id}/multi/regenerate`
- `GET /api/study-plans/{plan_id}/courses`
- `GET /api/study-plans/{plan_id}/schedule`
- `GET /api/study-plans/{plan_id}/progress`

## 知识图谱

- `POST /api/courses/{course_id}/knowledge-graph/jobs`
- `GET /api/courses/{course_id}/knowledge-graph/jobs/{job_id}`
- `GET /api/courses/{course_id}/knowledge-graph`
- `GET /api/courses/{course_id}/knowledge-graph/versions`
- `DELETE /api/courses/{course_id}/knowledge-graph/jobs/{job_id}`

## 头像与检索

- `PUT /api/auth/avatar`
- `DELETE /api/auth/avatar`
- `POST /api/search/semantic`

检索指标为 `cosine`，`similarity_score = 1 - distance`。
