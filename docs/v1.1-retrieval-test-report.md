## 未对真实用户数据执行的操作

出于安全约束，没有对项目真实 MySQL 或真实 `chroma_db` 执行 `--apply`，也没有删除或修改旧 collection。隔离 CLI 校验使用 `--skip-query` 避免下载真实模型；带查询的校验流程已由 fake embedding/fake Chroma 自动化测试覆盖。用户实际迁移时仍应在备份后运行不带 `--skip-query` 的完整校验。
