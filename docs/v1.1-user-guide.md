# 1.1 用户指南

## 资料引用

在“课程问答”选择课程后提问。回答中的 `[C1]` 等编号来自真实语义检索；回答下方“参考资料”可展开课程、资料、页码、片段序号、相似度和原文。切换历史会话后引用仍会恢复。没有足够相似的课程片段时不会强行给出引用。

## 综合规划

进入“学习计划 → 综合规划”，至少选择两门课程，分别设置优先级、截止日期和目标分钟数，再设置计划日期、每日容量及可学习星期。先“生成预览”查看容量、未排分钟和每日安排，确认后才创建真实任务。重新生成只替换该计划未完成的自动任务。

## 知识图谱

在课程卡片点击“知识图谱”。课程必须已有解析成功且完成向量化的资料，并且当前用户已配置可用大模型。生成状态按后台任务实时更新；点击节点或连线可查看资料、页码和证据片段。新版本失败不会覆盖旧的可用图谱。

## 用户头像

打开左下角“设置 → 个人头像”，上传 JPEG、PNG 或 WebP。服务端会压缩为不超过 256×256 的 WebP，并仅保存到当前账户。侧栏和课程问答中的用户消息会使用同一头像。

## 多文件上传

资料库可一次选择最多 10 个文件。点击上传前会显示待上传清单，可移除误选文件；上传后显示每个文件的进度与解析/向量状态。切换页面不会取消模块级上传队列。
